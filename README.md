##Description
An extension of a deep convolutional neural network (FER_DCNN) trained on the FER2013+ dataset for eight-class emotion classification: anger, contempt, disgust, fear, happiness, neutral, sadness, and surprise. We first reproduce the baseline results, then explore multiple targeted modifications to improve generalization, and finally propose and evaluate a transfer-learning approach using a pretrained ResNet18 as the improved method.

##Dataset
Dataset used: [FER2013Plus Dataset (Kaggle)](https://www.kaggle.com/datasets/subhaditya/fer2013plus)
